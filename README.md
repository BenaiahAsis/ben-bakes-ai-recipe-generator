# 🍪 Ben Bakes | AI-Powered Custom Cookie Architect

A full-stack web application that leverages Google's Gemini API to generate custom baking recipes based strictly on the ingredients a user currently has in their pantry. 

Built as a portfolio project to demonstrate API integration, strict prompt engineering, and database management.

## ✨ Features
* **Strict Prompt Engineering:** The AI is constrained by hard rules. It is forbidden from adding unlisted ingredients and is programmed to politely reject impossible combinations (e.g., trying to bake with just water and salt) while suggesting the bare minimum additions needed.
* **Markdown Parsing:** The raw Markdown text generated by the Gemini API is intercepted and parsed into beautiful, readable HTML on the frontend.
* **Persistent "Recipe Vault":** Successfully generated recipes are automatically saved to a MySQL database and displayed dynamically on the frontend.
* **Responsive UI:** Clean, modern, bakery-themed user interface built with Bootstrap 5, complete with asynchronous loading states for a polished UX.

## 🛠️ Tech Stack
* **Backend:** Laravel / PHP
* **Database:** MySQL (Eloquent ORM)
* **AI Engine:** Google Gemini 2.5 Flash API
* **Frontend:** Blade Templating, Bootstrap 5, Custom CSS/JS

## 🚀 How It Works
1. The user inputs their available ingredients into the UI.
2. The Laravel `RecipeController` validates the request and structures a highly-constrained prompt.
3. An HTTP POST request is fired to the Google Gemini API.
4. If successful, the JSON response is decoded, and the recipe string is extracted.
5. The original ingredients and the generated recipe are saved to the MySQL `recipes` table.
6. The view is returned, parsing the Markdown into HTML and retrieving the latest 5 entries from the database to display in the Recipe Vault.

## 💻 Local Setup
If you want to run this project locally:

1. Clone this repository.
2. Run `composer install` to grab the dependencies.
3. Duplicate `.env.example` and rename it to `.env`.
4. Generate an application key with `php artisan key:generate`.
5. Add your Google Gemini API key to the `.env` file: `OPENAI_API_KEY=your_key_here`
6. Set up your local MySQL database and run `php artisan migrate`.
7. Boot the server with `php artisan serve`.
